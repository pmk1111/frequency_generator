가청 주파수 테스트

Hz 자동 증가

사용 방법

1. 원하는 파형을 선택합니다.

2. 막대를 움직여서 주파수를 선택합니다.

3. 또는 원하는 주파수를 입력합니다.

4. 재생/정지 버튼을 눌러 주파수를 재생/정지할 수 있습니다.

5. 재생 중 막대를 조절해 출력 주파수를 조정할 수 있습니다.

6. 자동 증가 항목을 체크하면 초당 100hz씩 증가합니다.
